# UK data services
## Open access
For this project, I am using only open access data from the UK Data Service. Open data can be accessed without registration and is made available under open licences such as the Open Government Licence or Creative Commons, making it suitable for a school project.  [UK Data Service](https://ukdataservice.ac.uk/find-data/access-conditions/)

## Safeguarded
I am not using safeguarded data because it is subject to the End User Licence Agreement and additional restrictions. In particular, safeguarded data must not be shared or redistributed to anyone who is not an authorised user, and it must not be used with tools or methods that could expose or transfer the data improperly. These restrictions make safeguarded data unsuitable for this project.  [UK Data Service](https://ukdataservice.ac.uk/find-data/access-conditions/)

## Controlled
I am also not using controlled data because it is intended for very detailed, sensitive, or confidential information and can only be accessed by trained and accredited researchers through secure environments such as the UKDS SecureLab, SafePods, SafePoints, or the Safe Room.  [UK Data Service](https://ukdataservice.ac.uk/find-data/access-conditions/)
